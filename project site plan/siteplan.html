Trail Tracker — Site Plan

1. Site Name

Trail Tracker

This name was chosen because the website helps users track, find, and explore hiking trails easily. It is short, simple, and memorable, and it reflects the goal of guiding hikers to the best trails in Rwanda.

2. Site Purpose

The purpose of this website is to:

  • Provide a guide to the best hiking trails in Rwanda.
  • Help users filter and search trails by difficulty and length.
  • Offer safety tips and planning resources for beginner hikers.
  • Display trail locations using maps for easier navigation.

3. Scenarios

  1. "I am new to hiking. Which trails are safe for beginners in my area?"
  2. "How long is this trail and what should I carry with me before hiking?"

4. Color Scheme

Color Usage
Forest Green (#2d6a4f) Headings, borders, accents
Light Green (#d8f3dc) Backgrounds for highlight areas
Soft Mint (#74c69d) Table header background
Off White (#f2f9f5) Main page background

5. Typography

Heading Font: Trebuchet MS, sans-serif — used for all headings.

Body Font: Segoe UI, Arial, sans-serif — used for paragraphs and lists.

These fonts were selected because they are clean, readable, and professional, making the content easy to scan and comfortable to read on all devices.
