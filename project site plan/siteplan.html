Adventure Rwanda — Site Plan

1. Site Name

Adventure Rwanda

This name was chosen because the website promotes tourism and adventure activities in Rwanda, helping visitors explore the country’s natural beauty.

2. Site Purpose

The purpose of this website is to:

  • Provide information about tourist attractions in Rwanda.
  • Showcase adventure activities such as hiking, gorilla trekking, and cultural tours.
  • Help visitors plan trips and make reservations.

3. Scenarios

  1. How can a tourist find the best adventure places to visit in Rwanda?
  2. What activities are available for first-time visitors?

4. Color Schema

Color Usage
Blue (#0077b6) Headings and borders
Light Blue (#f0f8ff) Background color
Dark Blue (#002147) Main text color

5. Typography

Font Family: Arial, sans-serif

This font is used for all text to keep the website simple, readable, and professional.
